# API Advanced

Python functions that query the Reddit API to retrieve
subreddit data such as subscriber counts, hot posts,
and keyword frequencies.

## Files
- 0-subs.py: Returns the number of subscribers for a subreddit
- 1-top_ten.py: Prints the top 10 hot posts for a subreddit
- 2-recurse.py: Recursively returns all hot article titles
- 3-count.py: Recursively counts and sorts keyword occurrences
